# 📦 Return Data Analysis – SQL Project

## 🧠 Project Overview

This project analyzes product return behavior using SQL to uncover key drivers behind returns, including product categories, pricing strategy, discounts, logistics, customer demographics, and sustainability impact.

The goal is to identify patterns that contribute to high return rates and provide actionable business insights to reduce returns, improve profitability, and enhance customer experience.

---

## 📊 Dataset Overview

The dataset includes order-level return information with attributes such as:
- Product category
- Product price
- Discount applied
- Return status and reason
- Shipping method
- Customer demographics (age, gender)
- Financial metrics (profit impact, return cost)
- Sustainability metrics (CO2 emissions, packaging waste)

---

## 📈 Key Business Insights

### 💰 Financial Impact
- Overall return rate: **29%**
- Total profit impact: **$3.08M**
- Total return cost: **$290K**
- Average cost per return: **$200**

---

### 📦 Category Insights
- Clothing has the highest return rate at **37.43%**
- Electronics and Home Appliances also show elevated return rates (~25%)
- Books and Toys have the lowest return rates (~22–24%)

---

### 🔁 Return Drivers
- Defective products (382 cases)
- Changed Mind (379 cases)
- Wrong Item (348 cases)
- Size Issues (341 cases)

---

### 🚚 Logistics Insights
- Shipping method has minimal impact on return rates (~475–495 returns across all types)
- Average return time is ~31 days across all categories and reasons

---

### 👥 Customer Behavior
- Age groups show nearly identical return rates (~29%)
- Female customers (30.20%) return slightly more than male customers (27.72%)

---

### 💸 Pricing & Discount Impact
- High discounts lead to highest return rate (29.72%)
- High and low-priced items have higher returns than mid-range products
- Medium pricing performs best in terms of return stability

---

### 🌍 Sustainability Impact
- CO2 emissions from returns: **2,167.5**
- Packaging waste generated: **864.2**
- No CO2 savings or waste reduction observed

---

## 📌 Key Takeaways

- Returns are driven more by **product quality and expectation mismatch** than logistics or demographics.
- Clothing is the most critical category requiring intervention.
- Pricing strategy and excessive discounts contribute to higher returns.
- Sustainability impact of returns is significant and currently unmitigated.
- Operational improvements in fulfillment and product accuracy can reduce returns significantly.

---

## 🚀 Business Recommendations

- Improve product descriptions and sizing accuracy (especially Clothing)
- Reduce excessive discounting on high-return categories
- Strengthen quality control to reduce defective products
- Improve order fulfillment accuracy (reduce wrong item shipments)
- Introduce sustainability-focused return optimization strategies
